# django-z
简介：自定义django功能，优化admin后台功能，用于快速搭建后台服务

# 1. 项目依赖
```text
django == 3.2.22
python3 == 3.6.8
```

# 2. 下载该项目
```shell
git clone https://github.com/funtrue/django-z.git
```

# 2. 运行该项目
```shell
cd django-z

pip3 install -r requirements.txt
# pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
```
